# m-event
Management Event with React and AdonisJS

## setup
```
[1] create database
[2] input name database to .env file
```
## clone repository
```
$ git clone https://github.com/ibnusyawall/m-event
```
## install api
```
$ npm i -g @adonisjs/cli
$ cd m-event/event-api
$ npm i
$ adonis migrate:run
$ adonis serve --dev
```
## install app
```
$ cd m-event/event-app
$ npm i
$ npm start
```
